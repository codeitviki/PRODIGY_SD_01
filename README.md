# Temperature Conversion Program

This project is part of Task 01 for the Software Development Internship at Prodigy InfoTech.

## Description
The program converts temperatures between Celsius, Fahrenheit, and Kelvin.
The user provides a temperature value and its unit, and the program outputs the converted values.

## Technologies Used
- Python

## How to Run
1. Run the Python file
2. Enter temperature value
3. Enter unit (C / F / K)
4. View converted results
